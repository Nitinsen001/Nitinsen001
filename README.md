## 🌟 Featured Project

### 🧭 Campus Guide
**A real-time faculty locator with an interactive 3D campus model.**

> Students often waste time searching for faculty members because they don't know their real-time location — making repeated calls or asking around. **Campus Guide** solves this by predicting a faculty member's current location using timetable data, and suggesting alternate locations when they're unavailable.

**✨ Key Features**
- 🗺️ Interactive **3D campus model** — click any faculty member to view their live status
- 👤 Detailed profile cards: **Photo, Department, Designation, Cabin, Room No., Building, Floor**
- 📍 Selected room is **highlighted live** on the 3D model
- 🏛️ Special rooms labeled — **Principal Office, HOD Office, Laboratories**
- ⏱️ Significantly reduces unnecessary calls and saves students' time, improving campus navigation
